## Welcome to EKB's Github page!
EKB is DTU Ballerup's Electronics Club, and here you can find various public club projects, course material and club specific tools.

Since this club is a volunteer club, and this Github organisation and its repos are made by and for students, mistakes and errors can occur. Please contact us if you have issues or questions.
